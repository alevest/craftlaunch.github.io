⚡ How to Install
1
Download CraftLaunch_Setup.exe
Click the Download button above. The self-contained installer is ~28 MB and includes everything you need.
2
Run the installer
Double-click CraftLaunch_Setup.exe. Windows SmartScreen may show a warning — click "More info" → "Run anyway". This is normal for new unsigned apps.
3
Java is handled automatically
The installer checks for Java 17 and downloads it for you if it's missing. You don't have to do anything.
4
Launch and play
CraftLaunch opens automatically after install. Pick your Minecraft version, set your username, and click Play. That's it.
