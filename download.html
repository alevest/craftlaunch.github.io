How to Install

4 STEPS · ~2 MIN
1
Click the Download button above
The installer is 67 MB and fully self-contained — it includes the launcher, uninstaller, and everything else. No extra files needed.
2
Run CraftLaunch_Setup.exe
If Windows SmartScreen pops up, click "More info" then "Run anyway". This is normal for apps without a paid code-signing certificate — the file is safe.
3
Java is handled for you
The installer automatically detects whether Java 17 is on your machine. If it's missing, it downloads and installs it silently — you don't need to do anything.
4
Open CraftLaunch and play
CraftLaunch opens automatically after install. Select any Minecraft version, type a username, and hit Play. No account, no login, no subscription needed.
