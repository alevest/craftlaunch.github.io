⚡ How to Install
1
Click Download above
The self-contained installer is around 67 MB and includes everything — no extra files needed.
2
Run CraftLaunch_Setup.exe
If Windows SmartScreen appears, click "More info" then "Run anyway". This is normal for new apps without an expensive code signing certificate.
3
Java is handled automatically
The installer detects Java 17 and downloads it silently if it's missing. You don't need to do anything.
4
Launch and play
CraftLaunch opens after install. Pick your Minecraft version, type your username, and hit Play. That's it — no account, no subscription.
